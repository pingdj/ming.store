内建多聚合器 SWAP 交易

软件内直接完成链上交易，无需跳转外部 DEX。系统自动向 0x、ParaSwap、OKX DEX 等多个聚合器请求实时报价，自动选择最优路径。连接 MetaMask 或 Phantom 钱包后，实时显示余额，支持一键填入最大可交易数量。EVM 和 Solana 双链全面支持。

交易前强制安全检测

点击"确认并签名"后，系统自动调用 GoPlus API 对目标代币合约进行安全扫描。低风险代币静默放行；中风险代币弹出详细风险项（如合约未开源、税率过高）供您判断；高风险代币（如蜜罐、钓鱼地址）强制拦截并醒目警告。检测失败时提示用户自行决定，确保交易安全。

EVM + Solana 双链并行

地址标签管理 + GoPlus 风险分析

❓ 常见问题

您可能关心的问题，答案都在这里。
支持哪些公链？
目前支持 Ethereum、BSC、Polygon、Arbitrum、Base 五条 EVM 链，以及 Solana。后续将增加 Aptos、Sui 等非EVM链。
需要自己准备节点 RPC 吗？
不需要。系统内置了公共 RPC 和 WSS 节点，开箱即用。也支持自定义添加私有节点，提高稳定性。
AI 解读功能是否需要额外付费？
完全免费。系统已内置 DeepSeek API 密钥，无需您自行注册或充值。
如何获取软件和激活码？
请通过页面上的「联系作者」按钮获取下载链接和激活码。软件为绿色免安装版，支持 Windows 10/11。
数据会泄露吗？
所有监控数据（交易记录、余额快照）仅存储在您本地电脑的 SQLite 数据库中，不会上传到任何服务器，隐私安全有保障。
国内网络无法连接链上节点怎么办？
软件内置代理配置功能，您只需填写本机代理地址（如 http://127.0.0.1:7897），即可通过代理访问 RPC/WSS。
地址标签有什么用？
您可以为关注的地址（如交易所、项目方、个人钱包）添加自定义标签和颜色，日志中自动替换显示标签名。结合 GoPlus 风险分析，可快速识别恶意地址、合约地址，提升监控效率与安全性。
内建 SWAP 交易是否需要额外手续费？
软件本身不收取任何手续费。交易过程中产生的费用均为链上 Gas 费和 DEX 协议费用，与您在其他平台交易完全一致。系统会自动向多个聚合器比价，帮助您找到最优汇率。
安全检测会自动拦截所有高风险代币吗？
检测结果分级处理：低风险代币自动放行，不影响交易体验；中风险代币会弹出风险项目列表供您自行决定是否继续；高风险代币（如蜜罐、钓鱼地址）会强制拦截并醒目警告。您仍有最终决定权，但系统会在您冲动下单前为您提供关键的风险信息。
