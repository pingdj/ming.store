❓ 常见问题

您可能关心的问题，答案都在这里。
支持哪些公链？
目前支持 Ethereum、BSC、Polygon、Arbitrum、Base 五条 EVM 链，以及 Solana。后续将增加 Aptos、Sui 等非EVM链。
需要自己准备节点 RPC 吗？
不需要。系统内置了公共 RPC 和 WSS 节点，开箱即用。也支持自定义添加私有节点，提高稳定性。
AI 解读功能是否需要额外付费？
完全免费。系统已内置 DeepSeek API 密钥，无需您自行注册或充值。
如何获取软件和激活码？
请通过页面上的「联系作者」按钮获取下载链接和激活码。软件为绿色免安装版，支持 Windows 10/11。
数据会泄露吗？
所有监控数据（交易记录、余额快照）仅存储在您本地电脑的 SQLite 数据库中，不会上传到任何服务器，隐私安全有保障。
国内网络无法连接链上节点怎么办？
软件内置代理配置功能，您只需填写本机代理地址（如 http://127.0.0.1:7897），即可通过代理访问 RPC/WSS。
